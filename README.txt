TWOD Framework

Version 0000.00001

- By David López Meseguer

----------------------

A 2D framework to Animate graphics from the same .Png/.jpg file.

It crops the image into pieces and animate them on along the X axis of the original image.
Y axis elements are other also usable animations.
Features Animation System with built-in Events.


Main Functionality complete.

Working on Custom Editor and Inspectors tools.
